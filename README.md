# AI_Dev_LP

経営者向け「AI駆動開発教室」の集客用LP

---

## 概要

**目的**  
経営者向け「実践型・AI駆動開発教室」の認知理解と応募受付

**ターゲット**  
非エンジニアの経営者

**ターゲットニーズ/課題** 
AIを使えば、非エンジニアでもバイブコーディングができると聞いてワクワク、ソワソワしているが、何からどうはじめて良いか、分からない

**オファー内容**  
非エンジニアの経営者でも、バイブコーディングを始められるきっかけと補助を提供する実践型の開発教室
　- 初回お試しでは、**Claude Codeの環境構築**と、**自社サイトのLPを2時間で作成**するところまで体験可能 
　　- 講座後、**懇親会**も開催
　　　　- 懇親会では、バイブコーディングに長けた現役フルスタックエンジニアへ質問し放題。
    　　 今後、バイブコーディングやAIをどう会社経営に活かすか？質問やディスカッションも可能。
