# AI_Dev_LP

経営者向け「AI駆動開発教室」の集客用LP

---

## 概要

**目的**  
経営者向け「実践型・AI駆動開発教室」の認知促進と応募受付

**ターゲット**  
AIを使えば、非エンジニアでもバイブコーディングができると聞いてワクワクしている一方で、  
「始めるきっかけ」や「最初の補助」があれば一歩踏み出したいと感じている経営者

**オファー内容**  
非エンジニアの経営者でも、バイブコーディングを始められるきっかけと補助を提供する実践型講座です。  
初回お試しでは、**Claude Code の環境構築**と、**自社サイトのLPを3時間で作成**するところまで体験できます。  
さらに、講座後には**懇親会**も案内予定です。  
懇親会では、バイブコーディングに長けた現役フルスタックエンジニアへ質問し放題です。
